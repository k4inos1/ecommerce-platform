## 🚀 How I Use This Skill

When working on this ecommerce project, I apply the frameworks from this skill to:

1. **Product Research** — Before adding a new product to the MongoDB catalog, analyze competition, demand, and profit margins using the criteria above.

2. **Profit Validation** — Every product must have a minimum 30% margin. Use the profit calculator framework (price - product cost - shipping - fees = net profit).

3. **Listing Quality** — When creating product names and descriptions for the admin panel, optimize for search and conversion using the listing optimizer guidelines.

4. **Category Strategy** — Use niche analysis to decide which product categories to expand (currently: Laptops, Phones, Audio, Tablets, Wearables, Monitors, Accessories).

5. **Supplier Notes** — When suggesting new products, include estimated supplier cost from Alibaba so the admin can evaluate margins before adding to the store.

---

## 📝 Product Validation Checklist (for this project)

Before adding any product via the Admin Panel (`/admin/products`):

- [ ] Monthly demand > 1,000 units (market exists)
- [ ] Competition level: low or medium
- [ ] Net margin ≥ 30% after all costs
- [ ] Not highly seasonal (or planned ahead)
- [ ] Has real supplier source (Alibaba/local)
- [ ] Positive trend (not declining)
